# False Nine

False Nine is a curated archive of modern football: season chapters, core players, club timelines, competitions, transfers, and storylines from 2005 onward.


## Methodology

Top-four clubs are determined by final domestic league tables.

Champions League, World Cup, and Euro sections use final result records and finalist core-player selections.

Core players are curated by importance to the team/season, not by a single statistic. Appearances, reputation, awards, role, and historical significance all matter.

Transfers are selected for historical impact, not just fee size.

Ballon d’Or entries are aligned to the season that primarily shaped the award. Pandemic and future-award placeholders are written as notes so they do not break player pages.

Top-four league placements are entered as final standings for the five major leagues from 2005-06 through 2024-25. Serie A 2005-06 uses post-Calciopoli official placement, not the original table order.

Champions League finals are entered from UEFA historical final records.

Ballon d’Or entries use the award top-three by calendar year shown in that season chapter. The 2020 Ballon d’Or was canceled, so 2020-21 has no top-three list.

Golden Boot entries use European Golden Shoe winners and domestic league top scorers. Tied domestic top scorers are entered as separate rows.

Core players are curated for learning context. S tier means central identity/season-defining player; A tier means major core contributor. They are not exact statistical rankings.

International tournaments are attached to the season chapter that leads into or contains that tournament summer/window.

2025-26 includes completed club-season and Champions League results as of June 2026; the 2026 World Cup is marked Coming soon because it takes place in summer 2026.

Future releases should add one season JSON file at a time, then list it in data/index.json.

URL state uses hash routes like #season/2008-09, #player/Lionel%20Messi, and #club/Barcelona, so shared links work on static hosting.
